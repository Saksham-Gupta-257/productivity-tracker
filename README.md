# ⏱️ Productivity Tracker

A web-based application designed to help users track their tasks, manage their time, and boost daily productivity. This project features secure user authentication and real-time cloud data storage, ensuring that your productivity data is synced and accessible from anywhere.

**🔴 Live Demo:** [View the Live Project Here](https://saksham-gupta-257.github.io/productivity-tracker/)

## ✨ Key Features

* **User Authentication:** Secure sign-up and login functionality powered by Google Firebase Authentication.
* **Real-Time Database:** Tasks and productivity metrics are instantly saved and retrieved using Firebase Database.
* **Cloud Sync:** Your data is tied to your account, so you never lose your progress.
* **Responsive Design:** Optimized for both desktop and mobile viewing.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend as a Service (BaaS):** Google Firebase
  * *Firebase Authentication* (User management)
  * *Firebase Realtime Database / Firestore* (Data storage)
* **Hosting:** GitHub Pages
